# Personal Notes Web Application

This is a web-based personal notes management application that allows users to create an account, log in, and manage their own notes securely. The system is designed to help users organize notes efficiently using features like pinning, favourites, and filtering.

## Features
- User registration and login
- Create, edit, and delete notes
- Pin and unpin important notes
- Mark notes as favourite / unfavourite
- Filter and search notes
- User-specific data storage
- Clean and simple user interface

## Technology Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python
- Database: MySQL
- Server: XAMPP (Localhost)

## Database
The database file is included in this repository as `mini_notepad.sql`.  
It contains the required tables and structure to run the project locally.

## Project Purpose
This project was developed to demonstrate understanding of:
- Web application structure
- User authentication
- Database integration
- CRUD operations
- Real-world web development concepts

## Note
This project is configured to run on a local development environment using XAMPP. Environment configuration files are not included for security reasons.
