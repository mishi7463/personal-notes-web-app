# Personal Notes Web Application

This is a web-based notes management system where users can log in
and manage their personal notes.

## Features
- User authentication
- Create, edit, delete notes
- Pin / unpin notes
- Favourite / unfavourite notes
- Filter and search notes

## Tech Stack
- Backend: PHP / Python
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Server: XAMPP (Localhost)
